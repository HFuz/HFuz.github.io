Vanguard

By: Humza Fazal

Description

Vanguard is a new FPS game which is heavily inspired by games like Counter-Strike and Valorant. This is specifically a tactical FPS, which has many concepts that can be enhanced through VR. This kind of game is PVP (Player vs Player), it is meant to be played online against other players. There are different modes that the players can enjoy. Deathmatch is a simple free for all where players can shoot and defeat one another in a small arena. Despite the game being meant to be played online, there are still ways to play alone.
